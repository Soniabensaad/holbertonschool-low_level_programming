## Project Requirements
- **OS:** Ubuntu 14.04 LTS
- **gcc:** gcc 4.8.4 (C90)
- **make:** GNU Make 3.81

## File Descriptions

**4-Makefile:** makefile for executable `holberton`; rules: `all`, `clean`, `oclean`, `fclean`, `re`, variables: `CC`, `SRC`, `OBJ`, `NAME`, `RM`, `CFLAGS` `-Wall -Werror -Wextra -pedantic`
